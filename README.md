#  Fake News Headline Generator

##  Project Overview

Fake News Headline Generator is a fun Python project that generates random fictional news headlines by combining subjects, actions, and locations. The project demonstrates the use of Python lists, loops, user input handling, and the random module.

This project is designed for learning Python fundamentals and creating entertaining headline combinations.

---

##  Project Objectives

* Learn Python programming concepts.
* Understand list manipulation.
* Use the random module for content generation.
* Practice loops and conditional statements.
* Create an interactive command-line application.

##  How It Works

The program randomly selects:

* A Subject
* An Action
* A Place

and combines them into a humorous headline.

### Example Headlines

```text
BREAKING: virat kohli launches at red fort

BREAKING: a group of monkeys celebrates in pune metro station

BREAKING: auto rickshaw driver from delhi eats a plate of samosa
```
